# Cluster Configuration

## Directory Structure

- **`argocd-apps/`**: Contains chart templates and value files.
- **`sealed-secrets/`**: Stores sealed secrets.
- **`kube-prometheus/`**: Automatically generated by CI. See [CI Build info](https://github.com/Obmondo/k8id#ci-build-and-automatic-pull-requests).

## Files

- **`<cluster-name>-vars.jsonnet`** (e.g., `test.cluster.com-vars.jsonnet`): Used to generate the `kube-prometheus` configuration.
